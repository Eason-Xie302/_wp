學習 Markdown 非常簡單！它是一種輕量級的標記式語言，讓你可以透過在純文字中加入簡單的符號，輕鬆完成排版。許多寫作平台、筆記軟體（如 Notion、Obsidian）和開發者工具（如 GitHub）都廣泛使用它。

以下為你整理最常用、最核心的 Markdown 語法：

---

### 1. 標題 (Headings)
在文字前面加上 `#` 符號和一個「空格」。`#` 的數量代表標題的層級（最多支援到六級）。

**寫法：**
```markdown
# 這是一級標題 (最大)
## 這是二級標題
### 這是三級標題
```

### 2. 字體變化 (Emphasis)
利用星號 `*` 或波浪號 `~` 包裹文字來改變樣式。

**寫法：**
```markdown
**這是粗體**
*這是斜體*
***這是粗斜體***
~~這是刪除線~~
```
**效果：**
**這是粗體** | *這是斜體* | ***這是粗斜體*** | ~~這是刪除線~~

### 3. 列表 (Lists)
列表分為「無序列表」（只有圓點）和「有序列表」（有數字）。

**無序列表寫法：**（使用 `-` 或 `*` 加上空格）
```markdown
- 蘋果
- 香蕉
  - 芭蕉 (前面加兩個空格即可縮排產生次層級)
```

**有序列表寫法：**（使用數字、句號加上空格）
```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 4. 連結與圖片 (Links & Images)
這兩者的語法很像，圖片只比連結多了一個驚嘆號 `!`。

**連結寫法：** `[顯示文字](網址)`
```markdown
歡迎來到 [Google](https://www.google.com)
```

**圖片寫法：** `![替代文字](圖片網址)`
```markdown
![可愛的貓咪](https://example.com/cat.jpg)
```
> **提示：** 替代文字是在圖片失效時顯示的文字，也有助於網頁無障礙閱讀。

### 5. 引用 (Blockquotes)
當你需要引用別人的一段話時，可以在文字前加上大於符號 `>` 和空格。

**寫法：**
```markdown
> 這是一段引用的文字。
> 它可以有很多行。
```

### 6. 程式碼 (Code)
如果你需要標記程式碼或快捷鍵，可以使用反引號 `` ` ``（通常在鍵盤左上角 Esc 鍵下方）。

**行內程式碼：**（用單個反引號包住文字）
```markdown
你可以使用 `Ctrl + C` 來複製。
```

**多行程式碼區塊：**（用三個反引號包住上下，還可以指定語言來上色）
````markdown
```python
print("Hello, World!")
```
````

### 7. 分隔線 (Horizontal Rules)
連續輸入三個以上的連字號 `-` 或星號 `*` 就可以產生一條分隔線。

**寫法：**
```markdown
---
```

### 8. 表格 (Tables)
使用直線 `|` 來分隔欄位，使用連字號 `-` 來分隔標題與內容。

**寫法：**
```markdown
| 姓名 | 年齡 | 職業 |
| --- | --- | --- |
| 小明 | 25 | 工程師 |
| 小華 | 28 | 設計師 |
```

---

Markdown 的核心精神就是「易讀易寫」。一開始不熟練沒關係，多用幾次就會變成肌肉記憶了！
